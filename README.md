## Description

[Nest](https://github.com/nestjs/nest) repository.

## Installation

```bash
Run in console
$ npm install
```

## Configuration

```bash
Configure a .env file
PORT = 3000
USER_NAME = root
PASSWORD = 12345
DB_NAME = world
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
